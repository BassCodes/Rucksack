Single audio element

```html
<figure data-rsc>
	<figcaption>Song 1</figcaption>
	<audio controls src="media/example1.mp3"></audio>
</figure>
```

Multiple audio list

```html
<ol data-rsc>
	<li>
		Title 1
		<audio controls src="media/example1.mp3"></audio>
	</li>
	<li>
		Title 2
		<audio controls src="media/example2.mp3"></audio>
	</li>
</ol>
```
